NettyFinder 🚀

This script checks a Nigerian Telephone number and detects which network it belongs to.

Press F5 or Refresh this page to Get Started 😎
Enter number without (+234), e.g 08109445504 🔥
Number must be 11 digits and not less than. 🔥
Press F5 or Refresh this page to try another number 😎
